what i've been working on

weather project

This weather application fetches real-time weather information from over 200,000 cities using OpenWeatherMap APIs. It features a from that accepts a city from the user, and returns updated weather and a 5-day forecast. The application was initially built using HTML, CSS, and Javascript, but was later rebuilt using React.

react dictionary

This dictionary application features a from that accepts a word from the user, and returns the definition, related pictures, and adjectives. The website was built using React and APIs.

cyclistic case study

In this case study, I analyze historical data from a Chicago based bike-share company, Cyclistic, in order to identify trends in how casual riders and annual members use the service differently. The main tools I'm using are Excel, SQL and Tableau. Come back soon when the project is complete.
